🔗 Kết nối Google Sheets

Để sync dữ liệu với Google Sheets, bạn cần tạo một Google Apps Script. Làm theo các bước sau:

Bước 1: Mở Google Sheets → Extensions → Apps Script

Bước 2: Dán code sau vào editor:

function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSheet();
  var data = JSON.parse(e.postData.contents);
  data.forEach(function(row) {
    sheet.appendRow([
      row.date, row.topic, row.content,
      row.likes, row.comments, row.shares, row.status
    ]);
  });
  return ContentService.createTextOutput('OK');
}

function doGet(e) {
  var sheet = SpreadsheetApp.getActiveSheet();
  var data = sheet.getDataRange().getValues();
  return ContentService.createTextOutput(
    JSON.stringify(data)
  ).setMimeType(ContentService.MimeType.JSON);
}
